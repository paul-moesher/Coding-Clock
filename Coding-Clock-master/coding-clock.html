Paul's Coding Clock

Sometimes when you are coding it can be hard to stay focused. So I put this timer together, the idea is you set an amount of time that you want to code and stay focused for and then you also set yourself an amount of time that you can break for. Once the active timer expires the noise will sound and the other timer will start. I use this myself when coding to help keep on track. To change either the break time or the work time the clock must be paused. WHEN YOU INCREASE TIME YOU ARE INCREASING IT BASED OFF OF THE AMOUNT OF TIME YOU STARTED WITH NOT THE CURRENT TIME, THEREFORE CHANGING TIME WILL START A NEW SESSION.
